# Example Prompts

These example prompts demonstrate the kinds of requests the **3d-website-architect** skill handles. Each JSON file contains a structured prompt with metadata matching the eval test cases.

## Quick Start Prompts

### AI / SaaS Product Landing Page

> "Build a premium landing page for CodeLens AI, an AI-powered code review platform. Use Next.js, TypeScript, Tailwind CSS, and React Three Fiber. Include a 3D hero section with floating code blocks and an AI analysis sphere with bloom effects. Add smooth scroll animations with Framer Motion, a features grid, pricing section, and CTA."

See: [ai-code-review-landing.json](./ai-code-review-landing.json)

### Creative Studio Portfolio

> "Create a portfolio website for Nebula Creative, a design and motion studio. Use a warm dark theme with accent colors. Feature a morphing 3D organic blob as the hero, GSAP-powered text reveals on scroll, a filterable project gallery with hover tilts, and a full-screen contact section."

See: [creative-studio-portfolio.json](./creative-studio-portfolio.json)

### Cybersecurity / Enterprise

> "Design and build a cybersecurity product page for ShieldNet, an enterprise threat detection platform. Include a 3D interactive globe showing real-time connection arcs, animated stat counters, a dark enterprise theme with green accents, trust-building sections, and integration partner logos."

See: [cybersecurity-landing.json](./cybersecurity-landing.json)
